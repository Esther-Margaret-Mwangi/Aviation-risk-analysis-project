# Aviation-risk-analysis-project

Analysis of historical aviation accident data to identify low-risk aircraft for business investment using data cleaning, exploratory analysis, and interactive visualizations.

# Introduction

The purporse of this project was to help a company that plans to expand in to new industries to diversify its portfolio by purchasing and operating airplanes for commercial and private enterprises.
This project helps to identify the types of aircraft that have the least operational risk based on the dataset obtained from past aviation accidents.

# Data Understanding

The data used was obtained from Kaggle. The dataset contains records of aviation accidents that happened in the past. The data is stored in an excel file named flights.csv. The dataset has information about the accident date (acc.date), type of plane (type), aircraft registration (reg),
operator, fatalities (fat), location and damage outcomes (dmg). The dataset is perfect for this analysis since it potrays real word safety performance of different aircrafts.
